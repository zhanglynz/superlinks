  1. create a repo at Github
  2. create a project at RStudio
  3. run BookStarter::start_book()
  4. edit index.Rmd
  5. edit _bookdown.yml
  6. chapters 1, 2, 3, ….

7.2 My blog

  1. run ./R/to_have_a_new_post.R
  2. edit index.Rmarkdown: always adding “math: true” in the yaml header; always using <p>$$math formula$$</p>
  3. knit index.Rmarkdown
  4. manually dealing with in-line math formulas—using Find and Replace
  5. commit to Github
